# jh-skills

a directory of jason's personal skills

## list all skills

~~~bash
npx skills add jasonharmongit/jh-skills --list
~~~

## install a single skill

~~~bash
npx skills add jasonharmongit/jh-skills --skill <skill-name>
~~~

## install all skills

~~~bash
# install all skills to all detected agents
npx skills add jasonharmongit/jh-skills --all

# install all skills to a specific agent (example: cursor)
npx skills add jasonharmongit/jh-skills --skill '*' -a cursor
~~~
