# jh-skills

Personal skills published for the Skills CLI ecosystem.

## Install this repo

~~~bash
npx skills add jasonharmongit/jh-skills --list
~~~

## Install a single skill

~~~bash
npx skills add jasonharmongit/jh-skills --skill organize-elixir-functions
~~~

## Use the skill

After installation, ask your agent to use `organize-elixir-functions` when you want to:

- Organize functions in a specific file.
- Organize changed functions across the current branch.
- Preserve LiveView lifecycle ordering while reorganizing callbacks.
